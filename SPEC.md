# Focus Block 기획서

## 1. 배경 및 목적

Windows 11 시계 앱의 집중 세션 기능을 대체하는 커스텀 집중/휴식 타이머 앱.

### 기존 기능의 불편 사항

- 집중/휴식 시간을 세밀하게 설정할 수 없음 (50분, 55분 등 중간 단위 설정 불가)
- 총 시간 입력 방식만 지원하여 세션 수나 휴식 횟수를 알려면 직접 계산해야 함
- 집중 세션인지 휴식 세션인지 시각적으로 구분이 어려움 (작은 숫자 표시만 제공)

## 3. 핵심 기능 명세

### 3.1 세션 시간 설정

| 설정 항목 | 범위 | 단위 | 기본값 |
|----------|------|------|--------|
| 집중 시간 | 1분 ~ 120분 | 1분 | 25분 |
| 짧은 휴식 시간 | 1분 ~ 60분 | 1분 | 5분 |
| 긴 휴식 시간 | 1분 ~ 60분 | 1분 | 15분 |
| 긴 휴식 주기 | 2회 ~ 10회 | 1회 | 4회 |

---

### 3.2 세션 수 입력 방식 (3가지 중 택1)

사용자가 아래 3가지 방식 중 하나로 입력하면 나머지 값을 자동 계산하여 UI로 표시한다.

**총 수행 시간 입력**
```
입력: 총 수행 시간 (예: 120분)
계산: 세션 횟수, 짧은 휴식 횟수, 긴 휴식 횟수, 예상 종료 시각
```

**세션 횟수 입력**
```
입력: 집중 세션 횟수 (예: 4회)
계산: 총 소요 시간, 휴식 횟수, 긴 휴식 횟수, 예상 종료 시각
```

**휴식 횟수 입력**
```
입력: 짧은 휴식 횟수 (예: 3회)
계산: 집중 세션 횟수, 총 소요 시간, 긴 휴식 포함 여부, 예상 종료 시각
```

**세션 구성 미리보기 UI** — 입력값 변경 즉시 실시간 갱신

- 총 집중 시간 합계
- 총 휴식 시간 합계
- 긴 휴식 횟수
- 예상 종료 시각 (현재 시각 기준)
- 세션 흐름 시각화 (집중 · 짧은 휴식 · 긴 휴식을 아이콘 나열로 표현)

---

### 3.3 타이머 진행 화면

**상태별 시각적 구분** (와이어프레임 수령 후 확정)

| 상태 | 요구 조건 |
|------|----------|
| 집중 세션 | 집중 상태임을 한눈에 알 수 있는 강한 시각적 신호 |
| 짧은 휴식 | 집중과 확연히 다른 분위기 (색상, 레이아웃, 아이콘 등) |
| 긴 휴식 | 짧은 휴식과도 구분되는 별도 표현 |

**타이머 표시 요소**

- 남은 시간: 크고 가독성 높은 폰트로 표시
- 현재 상태 라벨: "집중 중" / "짧은 휴식" / "긴 휴식"
- 전체 진행도: 현재 N번째 세션 / 총 N번 세션 표시
- 세션 진행 바: 개별 세션 내 남은 시간 비율을 원형 또는 선형으로 표시
- 세션 맵: 전체 세션 순서를 아이콘 또는 점으로 나열, 완료된 세션 표시

---

### 3.4 타이머 제어

| 기능 | 설명 |
|------|------|
| 시작 | 설정 화면에서 진행 화면으로 전환 |
| 일시정지 / 재개 | 현재 세션 시간 정지 및 이어서 진행 |
| 다음 세션으로 | 현재 세션을 건너뛰고 다음 단계로 즉시 전환 |
| 처음으로 | 전체 세션을 초기화하고 설정 화면으로 복귀 |

---

### 3.5 알림

| 항목 | 내용 |
|------|------|
| 세션 전환 알림음 | 집중 → 휴식, 휴식 → 집중 전환 시 소리 재생 |
| Windows 토스트 알림 | 앱이 백그라운드에 있을 때도 상태 전환 알림 표시 |
| 알림음 on/off | 설정에서 토글 |

---

### 3.6 설정 영구 저장

| 항목 | 내용 |
|------|------|
| 저장 방식 | Electron `app.getPath('userData')` 기반 JSON 파일 |
| 저장 대상 | 집중/휴식/긴 휴식 시간 기본값, 긴 휴식 주기, 알림음 on/off |
| 적용 시점 | 앱 시작 시 자동 로드, 변경 시 즉시 저장 |

---

## 4. 화면 구성 (와이어프레임 확정 전 기준)

| 화면 | 역할 |
|------|------|
| 설정 화면 | 시간 설정 + 세션 수 입력 방식 선택 + 세션 구성 미리보기 |
| 진행 화면 | 타이머 + 현재 상태 표시 + 세션 맵 + 제어 버튼 |
| 설정 패널 | 기본값 저장, 알림 설정 |

---

## 5. 기술 구현 범위

| 영역 | 내용 |
|------|------|
| 타이머 엔진 | Electron Main Process에서 `setInterval` 기반 1초 단위 tick |
| IPC 통신 | Main ↔ Renderer 간 타이머 상태, 제어 명령 전달 |
| 애니메이션 | Framer Motion으로 상태 전환 시 화면 전환 효과 |
| 아이콘 | Lucide React |
| 스타일 | Tailwind CSS v4 |
